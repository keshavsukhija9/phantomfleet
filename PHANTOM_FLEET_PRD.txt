WHAT YOU ARE BUILDING (THE SHORT VERSION)
================================================================================

A Streamlit app that:
  1. Generates 50 fake shipments with live disruptions every few seconds
  2. Predicts which ones will fail (XGBoost, pre-trained, loads in 2s)
  3. Explains WHY using one Claude API call per flagged shipment
  4. Shows alternate routes through unused capacity on a Plotly map
  5. Auto-fixes low-risk shipments, asks human for high-risk ones
  6. Stores every intervention in memory and adjusts future scoring

That is the COMPLETE observe→reason→decide→act→learn loop.
Judges can see all 5 stages. Everything is real code.

  from dataclasses import dataclass, field
  from typing import Dict, List, Optional

  @dataclass
  class Shipment:
      id: str
      origin: str
      destination: str
      carrier: str
      eta_drift_pct: float        # how late vs schedule (%)
      carrier_reliability: float   # 0-1, rolling 30-day
      warehouse_pressure: float    # 0-1, downstream congestion
      weather_risk: float          # 0-1
      priority: str                # CRITICAL / HIGH / STANDARD
      status: str                  # HEALTHY / AT_RISK / RESCUED / FAILED
      failure_prob: float = 0.0
      intervention_id: Optional[str] = None

  @dataclass
  class Intervention:
      id: str
      shipment_id: str
      path: str                    # human-readable route
      predicted_eta_gain: float
      cost_delta_pct: float
      revival_prob: float
      execution: str               # AUTO / HUMAN / PENDING
      outcome: str = "PENDING"     # SUCCESS / FAILURE / PENDING
      causal_reason: str = ""
      score: float = 0.0

  @dataclass
  class AgentState:
      shipments: Dict[str, Shipment]       = field(default_factory=dict)
      risk_map:  Dict[str, float]          = field(default_factory=dict)
      interventions: Dict[str, Intervention] = field(default_factory=dict)
      pending_approvals: List[str]         = field(default_factory=list)
      tick: int                            = 0
      capacity_opportunities: List[dict]   = field(default_factory=list)
      episode_count: int                   = 0
      calibration_boost: Dict[str, float]  = field(default_factory=dict)
      # calibration_boost: maps shipment_id → score multiplier from memory

SECTION 3 : SYNTHETIC DATA GENERATOR (simulation/generator.py)
================================================================================

Build this in Hour 1. It is the heartbeat of the demo.

WHAT IT PRODUCES every tick (every 5 seconds real time = 30 min simulated):
  – 50 active shipments with randomized state
  – 3–5 disruptions injected per tick
  – Each shipment has the 8 features the model needs

KEY FEATURES TO GENERATE (only 8, not 22 — cuts training time to minutes):

  eta_drift_pct:          uniform(0, 40), spiked to 60+ during disruptions
  carrier_reliability:    beta(8, 2) baseline, drops to beta(2, 5) on degradation
  warehouse_pressure:     uniform(0.2, 0.6) baseline, spikes on congestion event
  weather_risk:           0.1 normally, 0.7-0.9 during weather event
  handoff_margin_hours:   uniform(1, 6), shrinks as delay accumulates
  downstream_critical:    int 0-3 (how many critical orders depend on this)
  priority_score:         CRITICAL=1.0, HIGH=0.67, STANDARD=0.33
  route_reliability:      average of 3 edge reliability scores

DISRUPTION SCHEDULE (hard-coded for demo reliability):
  Tick 3 : Carrier C4 degrades (reliability drops 0.85→0.25)
  Tick 5 : Weather event in Region B (weather_risk = 0.78)
  Tick 7 : Warehouse W3 congestion (warehouse_pressure = 0.91)
  Tick 10: Compound event (C6 breakdown + customs delay)
  Tick 15: Recovery begins (agent's learned interventions visible in metrics)

CODE SKELETON:

  import numpy as np
  import pandas as pd
  from dataclasses import asdict

  CARRIERS = ["C1","C2","C3","C4","C5","C6"]
  WAREHOUSES = ["W1","W2","W3","W4","W5"]
  PRIORITIES = ["CRITICAL","HIGH","STANDARD"]

  class SimulationEngine:
      def __init__(self, seed=42):
          np.random.seed(seed)
          self.tick = 0
          self.disruptions = {
              3:  {"type": "carrier_degradation", "target": "C4"},
              5:  {"type": "weather",  "region": "B"},
              7:  {"type": "warehouse_congestion", "target": "W3"},
              10: {"type": "compound", "targets": ["C6", "customs"]},
          }
          self.carrier_reliability = {c: 0.85 for c in CARRIERS}
          self.warehouse_pressure  = {w: 0.35 for w in WAREHOUSES}
          self.weather_risk = 0.1

      def step(self):
          self.tick += 1
          self._apply_disruptions()
          return self._generate_shipments()

      def _apply_disruptions(self):
          if self.tick in self.disruptions:
              d = self.disruptions[self.tick]
              if d["type"] == "carrier_degradation":
                  self.carrier_reliability[d["target"]] = 0.25
              elif d["type"] == "weather":
                  self.weather_risk = 0.78
              elif d["type"] == "warehouse_congestion":
                  self.warehouse_pressure[d["target"]] = 0.91
          # Gradual recovery after tick 12
          if self.tick > 12:
              self.weather_risk = max(0.1, self.weather_risk - 0.05)
              for k in self.carrier_reliability:
                  self.carrier_reliability[k] = min(0.85,
                      self.carrier_reliability[k] + 0.02)

      def _generate_shipments(self):
          shipments = []
          for i in range(50):
              carrier = np.random.choice(CARRIERS)
              warehouse = np.random.choice(WAREHOUSES)
              priority = np.random.choice(PRIORITIES, p=[0.10, 0.25, 0.65])
              base_drift = np.random.exponential(8)
              # Amplify drift if carrier is degraded
              if self.carrier_reliability[carrier] < 0.40:
                  base_drift *= 2.5
              s = {
                  "id": f"S{i:03d}",
                  "carrier": carrier,
                  "warehouse": warehouse,
                  "priority": priority,
                  "eta_drift_pct": min(base_drift, 80),
                  "carrier_reliability": self.carrier_reliability[carrier],
                  "warehouse_pressure": self.warehouse_pressure[warehouse],
                  "weather_risk": self.weather_risk,
                  "handoff_margin_hours": max(0.2, np.random.normal(3.0, 1.2)),
                  "downstream_critical": np.random.choice([0,1,2,3], p=[0.5,0.3,0.15,0.05]),
                  "priority_score": {"CRITICAL":1.0,"HIGH":0.67,"STANDARD":0.33}[priority],
                  "route_reliability": np.random.beta(7, 2),
              }
              shipments.append(s)
          return shipments

LABEL:
  failure = 1 if eta_drift_pct > 30 AND carrier_reliability < 0.5
              OR warehouse_pressure > 0.80
              OR handoff_margin_hours < 1.0
           0 otherwise
  (Simple heuristic label for synthetic data — good enough for demo.)

TRAIN SCRIPT (models/train.py):

  import numpy as np
  import pickle
  from xgboost import XGBClassifier
  from sklearn.model_selection import train_test_split
  from sklearn.metrics import roc_auc_score
  from simulation.generator import SimulationEngine

  # Generate 5000 training samples
  engine = SimulationEngine(seed=0)
  all_samples = []
  for _ in range(100):
      samples = engine.step()
      all_samples.extend(samples)

  df = pd.DataFrame(all_samples)
  features = ["eta_drift_pct","carrier_reliability","warehouse_pressure",
              "weather_risk","handoff_margin_hours","downstream_critical",
              "priority_score","route_reliability"]

  df["label"] = (
      ((df.eta_drift_pct > 30) & (df.carrier_reliability < 0.5)) |
      (df.warehouse_pressure > 0.80) |
      (df.handoff_margin_hours < 1.0)
  ).astype(int)

  X_train, X_test, y_train, y_test = train_test_split(
      df[features], df["label"], test_size=0.2, random_state=42)

  model = XGBClassifier(n_estimators=100, max_depth=4,
                         learning_rate=0.1, use_label_encoder=False,
                         eval_metric="logloss")
  model.fit(X_train, y_train)

  auc = roc_auc_score(y_test, model.predict_proba(X_test)[:,1])
  print(f"AUC: {auc:.3f}")  # should be > 0.88 with this setup

  with open("models/model.pkl", "wb") as f:
      pickle.dump(model, f)
  print("Model saved.")

INFERENCE WRAPPER (models/predict.py):

  import pickle, numpy as np, shap
  import pandas as pd

  FEATURES = ["eta_drift_pct","carrier_reliability","warehouse_pressure",
              "weather_risk","handoff_margin_hours","downstream_critical",
              "priority_score","route_reliability"]

  with open("models/model.pkl","rb") as f:
      MODEL = pickle.load(f)

  EXPLAINER = shap.TreeExplainer(MODEL)

  def predict(shipment_dict: dict) -> tuple[float, dict]:
      """Returns (failure_prob, top3_shap_features)"""
      row = pd.DataFrame([{k: shipment_dict[k] for k in FEATURES}])
      prob = float(MODEL.predict_proba(row)[0][1])
      sv = EXPLAINER.shap_values(row)[0]
      top3 = sorted(zip(FEATURES, sv), key=lambda x: abs(x[1]), reverse=True)[:3]
      return prob, {k: round(v, 3) for k, v in top3}

  from langgraph.graph import StateGraph, END
  from langgraph.checkpoint.memory import MemorySaver
  from agent.state import AgentState
  from agent.nodes import (observe, risk_assess, causal_reason,
                            plan, act, learn)

  def build_graph():
      wf = StateGraph(AgentState)
      wf.add_node("observe",       observe.run)
      wf.add_node("risk_assess",   risk_assess.run)
      wf.add_node("causal_reason", causal_reason.run)
      wf.add_node("plan",          plan.run)
      wf.add_node("act",           act.run)
      wf.add_node("learn",         learn.run)

      wf.set_entry_point("observe")
      wf.add_edge("observe",       "risk_assess")
      wf.add_conditional_edges(
          "risk_assess",
          lambda s: "causal_reason" if s.active_at_risk else END,
      )
      wf.add_edge("causal_reason", "plan")
      wf.add_edge("plan",          "act")
      wf.add_edge("act",           "learn")
      wf.add_edge("learn",         END)

      return wf.compile(checkpointer=MemorySaver(),
                        interrupt_before=["act"])  # human approval hook

  APP = build_graph()

NODE 2 — risk_assess.py
──────────────────────────────────────────────────────
  Runs XGBoost on every shipment. Flags at-risk. Attaches SHAP.

  from models.predict import predict
  from agent.state import AgentState

  THRESHOLD = 0.75

  def run(state: AgentState) -> AgentState:
      state.risk_map = {}
      state.active_at_risk = []
      state.shap_map = {}           # add shap_map to AgentState
      for sid, ship in state.shipments.items():
          prob, shap_top3 = predict(vars(ship))
          ship.failure_prob = prob
          state.risk_map[sid] = prob
          state.shap_map[sid] = shap_top3
          if prob >= THRESHOLD and sid not in state.interventions:
              state.active_at_risk.append(sid)
              ship.status = "AT_RISK"
      return state

NODE 3 — causal_reason.py
──────────────────────────────────────────────────────
  One Claude API call per at-risk shipment. Returns causal hypothesis.
  IMPORTANT: Call Claude ONCE per shipment. Do not loop inside the call.

  import os, json
  import anthropic
  from agent.state import AgentState

  CLIENT = anthropic.Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

  SYSTEM = """You are Phantom Fleet's Causal Reasoning Engine for logistics.
  Given shipment features and SHAP values, produce a 2-sentence causal
  hypothesis explaining WHY this shipment is at risk and what will happen
  if no action is taken. Be specific: cite the exact feature values.
  Respond only with valid JSON: {"hypothesis": "...", "primary_cause": "...",
  "confidence": 0.XX}
  primary_cause must be one of: CARRIER_DEGRADATION, WAREHOUSE_CONGESTION,
  WEATHER, DOWNSTREAM_PRESSURE, COMPOUND."""

  def run(state: AgentState) -> AgentState:
      state.causal_map = {}    # add causal_map to AgentState
      for sid in state.active_at_risk[:3]:   # cap at 3 to save API calls
          ship = state.shipments[sid]
          shap = state.shap_map.get(sid, {})
          prompt = f"""
  Shipment {sid} | Priority: {ship.priority}
  Failure probability: {ship.failure_prob:.1%}

  Feature values:
    eta_drift_pct       = {ship.eta_drift_pct:.1f}%
    carrier_reliability = {ship.carrier_reliability:.2f}
    warehouse_pressure  = {ship.warehouse_pressure:.2f}
    weather_risk        = {ship.weather_risk:.2f}

  Top SHAP drivers (what the model reacted to most):
    {shap}

  Produce your causal hypothesis as JSON."""

          resp = CLIENT.messages.create(
              model="claude-sonnet-4-20250514",
              max_tokens=200,
              system=SYSTEM,
              messages=[{"role":"user","content": prompt}]
          )
          try:
              result = json.loads(resp.content[0].text)
          except Exception:
              result = {"hypothesis": resp.content[0].text,
                        "primary_cause": "UNKNOWN", "confidence": 0.5}
          state.causal_map[sid] = result
      return state

NODE 4 — plan.py + capacity discovery
──────────────────────────────────────────────────────
  Finds alternate routes. Scores them. Simple — no A* needed at this scale.

  CAPACITY POOL (hard-code 10 opportunities for the demo):

  CAPACITY_POOL = [
      {"id":"CAP1","type":"truck","carrier":"C2","route":"W1→D1",
       "available_kg":450,"reliability":0.91,"eta_gain_hrs":3.5,"cost_delta":2.1},
      {"id":"CAP2","type":"flight","carrier":"AIR1","route":"BOM→MAA",
       "available_kg":800,"reliability":0.96,"eta_gain_hrs":6.2,"cost_delta":8.5},
      {"id":"CAP3","type":"van","carrier":"C3","route":"W2→D1",
       "available_kg":200,"reliability":0.88,"eta_gain_hrs":2.1,"cost_delta":1.5},
      {"id":"CAP4","type":"truck","carrier":"C5","route":"W3→D2",
       "available_kg":600,"reliability":0.84,"eta_gain_hrs":4.0,"cost_delta":3.2},
      {"id":"CAP5","type":"flight","carrier":"AIR2","route":"DEL→BOM",
       "available_kg":1000,"reliability":0.94,"eta_gain_hrs":5.5,"cost_delta":9.1},
  ]

  def score_path(cap, memory_boost=1.0):
      return (
          cap["reliability"] * 0.40
          + min(cap["eta_gain_hrs"] / 8.0, 1.0) * 0.35
          - min(cap["cost_delta"] / 10.0, 1.0) * 0.25
      ) * memory_boost

  from agent.state import AgentState, Intervention
  import uuid

  def run(state: AgentState) -> AgentState:
      for sid in state.active_at_risk[:3]:
          boost = state.calibration_boost.get(sid, 1.0)
          scored = [(score_path(c, boost), c) for c in CAPACITY_POOL]
          scored.sort(reverse=True)
          best_score, best = scored[0]
          intervention = Intervention(
              id=str(uuid.uuid4())[:8],
              shipment_id=sid,
              path=best["route"],
              predicted_eta_gain=best["eta_gain_hrs"],
              cost_delta_pct=best["cost_delta"],
              revival_prob=best["reliability"],
              execution="PENDING",
              causal_reason=state.causal_map.get(sid, {}).get("hypothesis",""),
              score=best_score,
          )
          state.interventions[intervention.id] = intervention
          state.shipments[sid].intervention_id = intervention.id
      return state

NODE 5 — act.py (with guardrails)
──────────────────────────────────────────────────────
  Executes low-risk actions. Escalates high-risk to human.

  AUTO_EXECUTE conditions (ALL must pass):
    1. score >= 0.70
    2. cost_delta_pct <= 5.0%
    3. priority != CRITICAL
    4. revival_prob >= 0.65

  from agent.state import AgentState

  def run(state: AgentState) -> AgentState:
      for iid, inv in state.interventions.items():
          if inv.execution != "PENDING":
              continue
          ship = state.shipments[inv.shipment_id]
          # Check all 4 guardrails
          guardrails = [
              inv.score >= 0.70,
              inv.cost_delta_pct <= 5.0,
              ship.priority != "CRITICAL",
              inv.revival_prob >= 0.65,
          ]
          if all(guardrails):
              inv.execution = "AUTO"
              ship.status = "RESCUED"
          else:
              inv.execution = "PENDING_HUMAN"
              if iid not in state.pending_approvals:
                  state.pending_approvals.append(iid)
      return state

NODE 6 — learn.py
──────────────────────────────────────────────────────
  Stores episodes in Chroma. Adjusts calibration_boost for similar future cases.
  This is what makes the scoring change over time — the actual learning.

  from memory.episodes import store_episode, get_boost
  from agent.state import AgentState

  def run(state: AgentState) -> AgentState:
      for iid, inv in state.interventions.items():
          if inv.outcome == "PENDING":
              continue
          # Store episode in Chroma
          store_episode({
              "id": iid,
              "shipment_id": inv.shipment_id,
              "path": inv.path,
              "revival_prob": inv.revival_prob,
              "cost_delta": inv.cost_delta_pct,
              "outcome": inv.outcome,
              "score": inv.score,
          })
          state.episode_count += 1
          # Update calibration boost for this carrier type
          ship = state.shipments.get(inv.shipment_id)
          if ship:
              boost = get_boost(ship.carrier, inv.outcome)
              state.calibration_boost[inv.shipment_id] = boost
      return state

SECTION 6 : VECTOR MEMORY (memory/episodes.py)

Simple Chroma store. 30 lines. Does exactly what's needed.

  import chromadb
  import json

  _client = chromadb.Client()
  _col = _client.get_or_create_collection("interventions")

  def store_episode(ep: dict):
      _col.add(
          ids=[ep["id"]],
          documents=[json.dumps(ep)],
          metadatas=[{
              "outcome": ep["outcome"],
              "path": ep["path"],
              "score": ep["score"],
          }]
      )

  def get_boost(carrier: str, last_outcome: str) -> float:
      """
      Query memory for similar past interventions involving this carrier.
      Return a score multiplier: >1 if historically successful, <1 if not.
      """
      try:
          results = _col.query(
              query_texts=[f"carrier {carrier}"],
              n_results=5,
              where={"outcome": {"$ne": "PENDING"}}
          )
          if not results["metadatas"][0]:
              return 1.0
          outcomes = [m["outcome"] for m in results["metadatas"][0]]
          success_rate = outcomes.count("SUCCESS") / len(outcomes)
          # Boost range: 0.8 (all failures) to 1.2 (all successes)
          return 0.8 + (success_rate * 0.4)
      except Exception:
          return 1.0

  col3.metric("Rescued", sum(1 for s in state.shipments.values()
                              if s.status == "RESCUED"))
  col4.metric("Episodes in Memory", state.episode_count)

  # ── Run one agent tick ───────────────────────────────────
  if st.button("Run Next Tick") or st.session_state.get("auto_run"):
      result = APP.invoke(state, config={"configurable":
                          {"thread_id": "demo"}})
      st.session_state.agent_state = result

  st.toggle("Auto-Run (every 5s)", key="auto_run")
  if st.session_state.get("auto_run"):
      time.sleep(5)
      st.rerun()

  # ── MAP PANEL ────────────────────────────────────────────
  col_map, col_risk = st.columns([2, 1])

  with col_map:
      st.subheader("Live Network Map")
      fig = build_map(state)   # function below
      st.plotly_chart(fig, use_container_width=True)

  with col_risk:
      st.subheader("Risk Feed")
      at_risk = sorted(
          [(sid, s) for sid, s in state.shipments.items()
           if s.failure_prob >= 0.50],
          key=lambda x: -x[1].failure_prob
      )
      for sid, ship in at_risk[:8]:
          bar_len = int(ship.failure_prob * 10)
          bar = "█" * bar_len + "░" * (10 - bar_len)
          priority_color = {"CRITICAL":"[CRITICAL]","HIGH":"[HIGH]","STANDARD":"[STANDARD]"}
          st.write(f"{priority_color[ship.priority]} **{sid}** "
                   f"`{bar}` {ship.failure_prob:.0%}")

  # ── REASONING PANEL ──────────────────────────────────────
  st.subheader("Agent Reasoning")
  causal_map = getattr(state, "causal_map", {})
  if causal_map:
      top_sid = state.active_at_risk[0] if state.active_at_risk else None
      if top_sid and top_sid in causal_map:
          c = causal_map[top_sid]
          st.info(f"**Shipment {top_sid}** | Cause: `{c.get('primary_cause','?')}`"
                  f" | Confidence: {c.get('confidence',0):.0%}")
          st.write(c.get("hypothesis",""))
          shap = getattr(state, "shap_map", {}).get(top_sid, {})
          if shap:
              st.caption("Top SHAP drivers: " + str(shap))
  else:
      st.success("No shipments at risk this tick.")

  # ── ESCALATION CARDS ─────────────────────────────────────
  if state.pending_approvals:
      st.subheader("Human Approval Required")
      for iid in state.pending_approvals[:]:
          inv = state.interventions.get(iid)
          if not inv:
              continue
          ship = state.shipments.get(inv.shipment_id)
          with st.expander(f"Shipment {inv.shipment_id} "
                           f"| {ship.priority} | Score: {inv.score:.2f}",
                           expanded=True):
              st.write(f"**Proposed path:** {inv.path}")
              st.write(f"**ETA improvement:** +{inv.predicted_eta_gain:.1f}h")
              st.write(f"**Cost increase:** +{inv.cost_delta_pct:.1f}%")
              st.write(f"**Revival probability:** {inv.revival_prob:.0%}")
              st.write(f"**Why:** {inv.causal_reason}")
              c1, c2 = st.columns(2)
              if c1.button(f"Approve", key=f"app_{iid}"):
                  inv.execution = "HUMAN_APPROVED"
                  ship.status = "RESCUED"
                  state.pending_approvals.remove(iid)
                  st.rerun()
              if c2.button(f"Reject", key=f"rej_{iid}"):
                  inv.execution = "REJECTED"
                  state.pending_approvals.remove(iid)
                  st.rerun()

  # ── METRICS CHART ─────────────────────────────────────────
  with st.expander("Learning Progress"):
      if state.episode_count > 0:
          st.write(f"Episodes stored in memory: **{state.episode_count}**")
          boosts = state.calibration_boost
          if boosts:
              import pandas as pd
              df = pd.DataFrame(list(boosts.items()),
                                columns=["Shipment","Score Multiplier"])
              st.bar_chart(df.set_index("Shipment"))
              st.caption("Score multiplier > 1.0 = memory boosting this path type."
                         " < 1.0 = memory penalizing based on past failures.")

  # ── MAP BUILDER ──────────────────────────────────────────
  def build_map(state: AgentState):
      # India-scale fake coordinates for 5 nodes
      NODES = {
          "W1": (19.0760, 72.8777),   # Mumbai
          "W2": (28.6139, 77.2090),   # Delhi
          "W3": (13.0827, 80.2707),   # Chennai
          "D1": (12.9716, 77.5946),   # Bangalore
          "D2": (22.5726, 88.3639),   # Kolkata
      }
      fig = go.Figure()
      # Draw node markers
      for name, (lat, lng) in NODES.items():
          fig.add_trace(go.Scattermapbox(
              lat=[lat], lon=[lng], mode="markers+text",
              marker=dict(size=18, color="steelblue"),
              text=[name], textposition="top right",
              name=name, showlegend=False
          ))
      # Draw shipment dots
      for sid, ship in state.shipments.items():
          if ship.failure_prob < 0.50:
              continue
          lat = NODES["W1"][0] + (hash(sid) % 100)/500
          lng = NODES["W1"][1] + (hash(sid) % 100)/500
          color = "red" if ship.failure_prob > 0.75 else "orange"
          size = 10 if ship.priority != "CRITICAL" else 16
          fig.add_trace(go.Scattermapbox(
              lat=[lat], lon=[lng], mode="markers",
              marker=dict(size=size, color=color, opacity=0.8),
              name=sid, showlegend=False,
              hovertext=f"{sid}: {ship.failure_prob:.0%} risk"
          ))
      # Draw rescue paths for active interventions
      for iid, inv in state.interventions.items():
          if inv.execution in ["AUTO","HUMAN_APPROVED"]:
              parts = inv.path.split("→")
              if len(parts) >= 2:
                  src = parts[0].strip()
                  dst = parts[-1].strip()
                  if src in NODES and dst in NODES:
                      fig.add_trace(go.Scattermapbox(
                          lat=[NODES[src][0], NODES[dst][0]],
                          lon=[NODES[src][1], NODES[dst][1]],
                          mode="lines",
                          line=dict(width=3, color="limegreen"),
                          name="Rescue Path", showlegend=False
                      ))
      fig.update_layout(
          mapbox=dict(style="open-street-map",
                      center=dict(lat=20.5937, lon=78.9629), zoom=4),
          margin=dict(l=0, r=0, t=0, b=0), height=400
      )
      return fig
